# @masekin/ui

Shared UI package for Masekin apps.

## Attribution

This package is a shared wrapper/customization layer built on top of
[shadcn/ui](https://ui.shadcn.com/) style components and related open-source UI libraries
(such as Radix UI and Lucide).

## Release Workflow

- Changelog: `CHANGELOG.md`
- Release guide: `RELEASING.md`
- Pre-release gate:
  - `npm run check:release`

## Install (Git tag)

```bash
npm i github:Ahbapx/masekin-ui#v0.1.2
```
